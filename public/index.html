Make a Quiz!

Making a quiz is simple:

  1. Fill out the quiz title and description.
  2. Add some questions.
    1. Click the "Add New Question" button.
    2. Fill out a title for the new question.
    3. Fill out the question's text (the actual question you want your visitors to be asked).
    4. Add some answers for that question.
      1. Fill out the text for each answer; Each answer is a choice your users can click on during the quiz.
      2. Specify some attribute modifiers to be associated with each answer.
        • Attribute modifiers attempt to associate an answer with one or more personality traits.
        • Visitors taking your quiz will start out with blank personality attributes. We'll keep track of each answer they select while taking your quiz, and its associated personality attributes.
        • When a visitor is finished with your quiz, we'll sum up all the personality attributes for all their chosen answers, and display the quiz result that most closely matches them.
        • Example: You might add a question with the text "What would you do if you were hungry?"
          • You might add the answer "I would eat a sandwich" with attribute modifiers "responsibility=+5;fun=-5;" to indicate a boring choice.
          • You might also add a second answer "I would eat ice cream" with modifiers "responsibility=-5;fun=+10;" to indicate a fun choice.
  3. Add some quiz results.
    • Fill out the title and description for each quiz result character.
    • Fill out the personality attributes for this quiz result character too.
    • When a visitor takes your quiz, their personality (based on the answers they chose as explained above) will be matched up with the most similar quiz result character here, and become the result of their quiz.
